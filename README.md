# Ataxx AI

基于Ataxx规则的智能AI系统，采用位板技术优化棋盘表示，结合搜索算法与评估函数实现高效决策。

在Botzone平台中排名30/1107名。

## 游戏简介

同化棋是一种二人棋盘游戏，黑白双方交替在7x7的棋盘上移动棋子。这个游戏与国际通行的同化棋规则基本一致，并增加了一些特殊规则。

### 游戏规则

1. 游戏在7x7的棋盘上进行，初始状态为双方在对角线上各有2枚棋子
2. 玩家轮流移动棋子，每次必须移动一枚己方棋子
3. 移动方式有两种：
   - **复制**：移动到以当前棋子为中心的3x3范围内（8个相邻位置），棋子会复制到新位置
   - **跳跃**：移动到3x3范围外、5x5范围内的位置（16个位置），棋子会移动到新位置
4. 每次落子后，新位置的棋子会将周围8个相邻位置的敌方棋子同化为己方棋子
5. 如果没有合法移动，游戏将继续，将所有空位算作有子可下一方的棋子
6. 游戏在以下情况下结束：
   - 一方棋子数为0时，对方获胜
   - 回合数达到400时，棋子多者获胜
   - 双方都无法移动时，棋子多者获胜

### 特殊规则

- **局面循环**：如果某一步使得局面回到曾经出现过的局面，对方将不能走出和当时一样的一步
- **回合限制**：游戏在400回合后强制结束，计算双方棋子数决定胜负

## 功能特点

- 人机对战模式，玩家可选择执黑或执白
- 强大的AI对手，基于极小极大搜索框架，集成Alpha-Beta剪枝与迭代加深算法
- 清晰的命令行界面，显示棋盘状态和游戏信息
- 完整的输入验证和错误处理
- 支持Windows系统（UTF-8编码）

## 编译和运行

### 环境要求

- C++11或更高版本编译器
- CMake 3.10或更高版本
- Windows系统（当前版本针对Windows优化）

### 构建步骤

```bash
# 克隆项目
git clone <repository-url>
cd ataxx-ai

# 使用CMake构建
mkdir build
cd build
cmake ..
cmake --build .

# 运行程序
./ataxx_ai.exe
```

### 直接运行

如果系统中已有编译好的可执行文件，可以直接运行：

```bash
./ataxx_ai.exe
```

## 游戏操作

1. 启动程序后，选择执黑棋(●)或白棋(○)
2. 系统会显示当前棋盘状态
3. 根据提示选择要移动的棋子和目标位置
4. AI会自动进行下一步移动
5. 游戏结束后会显示胜负结果

## 技术实现

- **位板技术**：使用位运算优化棋盘状态表示和操作，提高计算效率
- **极小极大搜索框架**：实现Minimax算法和Alpha-Beta剪枝提高AI决策效率
- **迭代加深算法**：结合迭代加深搜索，在时间限制内获得最优解
- **定制化评估函数**：融合棋子数量与位置权重的启发式评估函数优化AI棋力
- **时间限制控制**：使用时间限制确保AI在合理时间内做出决策
